# Ecommerce SQL Project

## 📌 Description
This project is a simple ecommerce database system for learning SQL.  
It includes customers, products, orders, and data analysis queries.

---

## 🧱 Tables
- Customers
- Products
- Orders
- Order_Items

---

## 🔗 Relationships
- One customer → many orders  
- One order → many order items  
- One product → many order items  

---

## 📊 Features
- Total sales per customer
- Best selling products
- Sales by category
- Order statistics

---

## 🛠️ Tools
- SQL (MySQL / SQL Server)
- Git & GitHub

---

## 🚀 How to run
1. Run schema.sql  
2. Run data.sql  
3. Run queries.sql
